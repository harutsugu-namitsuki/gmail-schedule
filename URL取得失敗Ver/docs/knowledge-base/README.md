# Knowledge Base — 設計と検討資料

本リポジトリにおける**ナレッジベース**は、トップファームのメール/記事から抽出したインサイトを永続的に蓄積する仕組みであり、以下の構成で実現している。

## 構成要素

| 要素 | 場所 | 役割 |
|---|---|---|
| **ナレッジベース本体（データ）** | [`/insights/`](../../insights/) | 1ファイル＝1インサイトとして個別に蓄積される実データ。日次タスクが自動生成 |
| **インサイトのフォーマット定義** | [`/templates/insight-universal.md`](../../templates/insight-universal.md) | Notion / Obsidian 両対応のユニバーサル設計テンプレート |
| **設計書（思想・運用ロードマップ）** | [`/docs/インサイトデータベース設計書.md`](../インサイトデータベース設計書.md) | なぜこのフォーマットか、どう運用していくか |
| **検討資料（歴史的記録）** | [`./検討資料/`](./検討資料/) | フォーマット採用前のプロダクト比較・代替案の評価記録 |

## 検討資料について

`./検討資料/` の中身は**フォーマット採用検討時の比較資料**であり、**現在は使用していない**。

- [ナレッジベース採用プロダクト比較.md](./検討資料/ナレッジベース採用プロダクト比較.md) — Notion / Obsidian / プレーンの3案を比較し、ユニバーサル設計（プレーン版）を選定するに至った評価マトリクス
- `insight-notion.md` / `insight-obsidian.md` / `insight-plain.md` — 3案それぞれの試作テンプレート（最終的に統合され `templates/insight-universal.md` となった）

意思決定の経緯を残すための歴史的記録として保持しているが、運用には参照しない。
